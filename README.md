# RabahDj

شبكة اجتماعية محلية بتصميم احترافي يشبه فيسبوك، تعمل بالكامل عبر شبكة **واي فاي محلية بدون إنترنت**. الأجهزة المتصلة بنفس الشبكة تتبادل المنشورات والإعجابات والتعليقات في الوقت الفعلي.

## كيف تعمل الفكرة؟

نظراً لأن تطبيقات React Native لا تدعم اكتشاف الأجهزة والتواصل المباشر (Peer-to-Peer) بدون إضافات أصلية معقدة، اعتُمد الحل الأكثر عملية واستقراراً:

- **جهاز واحد** على الشبكة (لابتوب أو هاتف بصلاحيات متقدمة) يشغّل **سيرفر Node.js محلي** (مجلد `server/`).
- **بقية الأجهزة** تفتح تطبيق RabahDj (مجلد `app/`) وتتصل بعنوان الـ IP المحلي لهذا السيرفر.
- كل التواصل (منشورات، إعجابات، تعليقات، من هو متصل الآن) يمر عبر هذا السيرفر بتقنية Socket.io في الوقت الفعلي.
- **لا حاجة لإنترنت إطلاقاً** — فقط اتصال كل الأجهزة بنفس شبكة الواي فاي (راوتر منزلي، أو نقطة اتصال Hotspot من أحد الهواتف).

## هيكل المشروع

```
RabahDj/
├── server/          # السيرفر المحلي (Node.js + Express + Socket.io)
│   ├── index.js
│   └── package.json
└── app/             # تطبيق الموبايل (React Native + Expo)
    ├── App.js
    ├── app.json
    └── src/
        ├── context/SocketContext.js
        ├── screens/ (تسجيل الدخول، الرئيسية، الملف الشخصي)
        ├── components/ (بطاقة منشور، إنشاء منشور)
        └── theme/colors.js
```

## خطوات التشغيل

### 1) تشغيل السيرفر (على جهاز واحد فقط، مثل لابتوب)

```bash
cd server
npm install
npm start
```

سيظهر في الطرفية عنوان مثل:
```
عنوان الشبكة المحلية: http://192.168.1.10:4000
```
احتفظ بهذا العنوان (IP)، فهو ما ستدخله كل الأجهزة الأخرى.

### 2) تشغيل التطبيق على كل جهاز

```bash
cd app
npm install
npm start
```

ثم افتح التطبيق عبر Expo Go على الهاتف (يجب أن يكون الهاتف متصلاً بنفس شبكة الواي فاي التي عليها السيرفر)، وأدخل:
- **اسمك**
- **عنوان IP** الذي ظهر في خطوة السيرفر (مثال: `192.168.1.10`)

بعد الاتصال، ستظهر لك الخلاصة الرئيسية وتقدر تنشر، تعجب بالمنشورات، وتعلّق، وكل هذا يظهر فوراً عند بقية المستخدمين المتصلين.

## المميزات الحالية
- تصميم احترافي مستوحى من فيسبوك (شريط المتصلين الآن، بطاقات منشورات، إعجابات وتعليقات)
- تحديثات فورية عبر Socket.io بدون الحاجة لتحديث الصفحة
- إرفاق صور مع المنشورات
- عرض عدد المتصلين حالياً بالشبكة

## بناء ملف APK لتثبيته مباشرة على الهاتف

الطريقة الرسمية لبناء APK من مشروع Expo هي عبر **EAS Build**. الخطوات:

```bash
cd app
npm install -g eas-cli
eas login              # أنشئ حساباً مجانياً على expo.dev إذا لم يكن لديك
eas build:configure    # سيتعرف تلقائياً على ملف eas.json الموجود بالمشروع
eas build -p android --profile preview
```

بعد انتهاء البناء (يستغرق عادة بضع دقائق على سيرفرات Expo)، ستحصل على **رابط تحميل مباشر لملف APK**. حمّله على هاتفك وثبّته (فعّل خيار "السماح بالتثبيت من مصادر غير معروفة" في إعدادات أندرويد).

> ملاحظة: عملية البناء نفسها تحتاج إنترنت لأنها تتم على سيرفرات Expo، لكن التطبيق بعد التثبيت يعمل بالكامل محلياً عبر الواي فاي كما هو مخطط له، بدون أي حاجة للإنترنت أثناء الاستخدام.

### بديل: بناء محلي بدون سيرفرات Expo
إذا كان لديك Android Studio مثبتاً، يمكنك البناء محلياً بدون إنترنت خارجي:
```bash
npx expo prebuild -p android
cd android
./gradlew assembleRelease
```
ستجد ملف الـ APK الناتج في:
`android/app/build/outputs/apk/release/app-release.apk`
